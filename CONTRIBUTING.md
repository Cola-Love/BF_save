# Contributing to BF_save

感谢你考虑为 BF_save 项目做出贡献！

## 如何贡献

### 报告 Bug

如果你发现了 Bug，请在 GitHub Issues 中创建一个 Issue，并包含以下信息：

- Bug 的详细描述
- 复现步骤
- 使用的 Python 版本和操作系统
- 相关错误日志

### 提交代码

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交你的更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建一个 Pull Request

### 代码规范

- 遵循 PEP 8 代码风格
- 保持代码简洁，避免不必要的抽象
- 提交信息使用中文或英文均可，但需清晰描述变更内容

### 测试

- 在提交 PR 前，请确保使用不同的存档文件测试你的修改
- 确保回退兼容性，勿破坏现有功能的存档解析

## 开发方向

欢迎以下类型的贡献：

- 支持更多 Unity BinaryFormatter 记录类型
- 改进 Web 界面的用户体验
- 添加更多游戏数据的映射表 (data.py)
- 性能优化
- 文档改进
